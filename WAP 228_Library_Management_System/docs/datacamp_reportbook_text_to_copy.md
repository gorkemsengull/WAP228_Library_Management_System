## General Description
During the online workplace training process, I completed DataCamp courses assigned by the department. These courses helped me improve my basic knowledge in data literacy, Python programming, Java programming, SQL databases, and GitHub usage. I also developed a small software project by using Python, SQLite, and GitHub.

## Data Literacy
In the Data Literacy track, I learned the basic concepts of data, data-driven decision making, data types, and how data can be used in real-world business problems. This training helped me understand how to interpret data correctly and how to ask meaningful questions before starting an analysis.

## Python Data Fundamentals
In the Python Data Fundamentals track, I studied Python basics, variables, data types, lists, functions, and basic data manipulation. This track helped me understand how Python can be used for problem solving and simple software development.

## Java Fundamentals
In the Java Fundamentals track, I learned basic Java syntax, variables, conditions, loops, methods, and object-oriented programming concepts. This course helped me compare Java and Python programming structures.

## SQL Data Fundamentals
In the SQL Data Fundamentals track, I learned database concepts, tables, primary keys, foreign keys, SELECT queries, filtering, JOIN operations, and grouping data. This knowledge was useful while designing the database of my project.

## GitHub Foundations
In the GitHub Foundations track, I learned the basic concepts of Git and GitHub, repositories, commits, branches, and project sharing. I used GitHub to publish and document my project.

## Project Work
For the project phase, I developed a Library Management System. The system allows adding books, adding members, borrowing books, returning books, and listing borrowing records. I used Python for the application logic and SQLite for the database.

## Conclusion
This online training process helped me improve my programming, database, and version control skills. I gained practical experience by completing DataCamp tracks and developing a small software project related to these topics.
